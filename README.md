# Intelligent Route Planner using Graph Algorithms

## Overview

This project implements an intelligent route planner using graph algorithms to calculate optimized routes between nodes while analyzing performance and route characteristics.

## Features

- Dijkstra shortest path algorithm
- BFS traversal
- DFS traversal
- Route visualization using graphs
- Traffic simulation
- Execution time analysis
- Node connectivity analysis
- CSV export functionality
- Route statistics generation
- Weighted graph implementation

## Technologies Used

- Python
- NetworkX
- Matplotlib
- Pandas
- Heapq
